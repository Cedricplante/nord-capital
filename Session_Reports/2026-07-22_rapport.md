## Décisions importantes
- **Une erreur de commit/push découverte en cours de session** : les 4 premiers fixes de l'audit avaient été édités dans les fichiers mais jamais commit/push — Vercel tournait encore sur l'ancien code. Rappel pour les prochaines sessions : toujours vérifier `git status`/`git diff origin/main` avant de conclure qu'un fix "ne marche pas".
- **Reconstruction historique plutôt que suppression** : face aux valeurs fausses de juin-juillet, Cédric a choisi de recalculer depuis les vraies données (trades + prix historiques) plutôt que d'effacer les lignes ou de deviner un chiffre.
- **Claude in Chrome comme contournement réseau** : le sandbox bash a un proxy qui bloque Yahoo Finance/CoinGecko, et `mcp__workspace__web_fetch` ne extrait pas le JSON brut de ces APIs. Naviguer directement vers l'URL JSON avec Claude in Chrome puis `get_page_text` fonctionne bien — à réutiliser si besoin de données de marché historiques à l'avenir.
